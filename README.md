# To-Do Lists

A simple and efficient to-do list application built with React, Tailwind CSS, and Vite.

## How It Works

- **Add Tasks**: Create new to-do items with a clean, intuitive interface
- **Manage Tasks**: Mark tasks as complete, edit, or delete them
- **Persistent Storage**: Tasks are saved and persist across sessions
- **Responsive Design**: Works seamlessly on desktop and mobile devices

## Tech Stack

- **React** - UI library for building interactive components
- **Vite** - Fast build tool and development server
- **Tailwind CSS** - Utility-first CSS framework for styling
- **ESLint** - Code quality and consistency

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start the development server:
   ```bash
   npm run dev
   ```

3. Open your browser and navigate to the displayed local URL

## Build for Production

```bash
npm run build
```
